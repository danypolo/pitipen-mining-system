Mejora del comportamiento tras ALT+TAB (ventana vuelve correctamente)
Corrección de overlay “always on top”
Ajustes visuales (texto “Creado por…” ya no se corta)
Mostrado de la versión actual en pantalla (v1.4)
🔄 Sistema de actualizaciones
Implementado sistema de comprobación de versión online
Lectura automática de version.json desde GitHub
Notificación cuando hay nueva versión disponible
Visualización de cambios incluidos en la actualización
🛠️ Instalación y distribución
Mejor integración en .exe (incluyendo hotkeys)
Corrección de problemas de permisos (config.json fuera de Program Files)
Instalador optimizado usando estructura completa de dist
Mejor compatibilidad con librería keyboard

v1.4 — Usability, OCR and system improvements

Release date: April 2026
Author: danypolo

✨ New in this version
🧠 Improved and configurable OCR
More stable and accurate OCR system
Added user-configurable OCR sensitivity
Better detection of small HUD numbers
Improved handling of dynamic HUD colors (not only yellow)
Better parsing of values like 51,000
More robust against visual noise

Automatic log cleanup on startup
Improved OCR error handling
Fixed conflicts between modes (material / hand)

Added global hotkey for calibration
Removed calibration confirmation popup
Improved ALT+TAB behavior (window restores correctly)

Display of current version (v1.4)

Online version checking implemented

Displays changelog inside the app

Fixed permission issues (no writing in Program Files)
